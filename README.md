## 🚀 Series: Java 25 "Zero to Hero" — พลิกโฉมการเขียนโค้ดปี 2026

### 📦 Module 1: ลบภาพจำ Java แบบเดิมๆ (Modern Foundation)

*เน้นความง่ายและฟีเจอร์ใหม่ที่ทำให้ Java เขียนสนุกเหมือนภาษาวัยรุ่น*


- Java 25 "Zero to Hero"  **EP  1 :** เลิกเขียน `public static void main` ยาวๆ ด้วยฟีเจอร์ "บรรทัดเดียวรันได้"
- **EP 2:** JShell: ลองโค้ด Java แบบไม่ต้องสร้างโปรเจกต์ (เหมือน Python/Node!)
- **EP 3:** ใช้ `var` ยังไงให้ดูหล่อ? ลด Code ขยะแต่ยัง Safe เหมือนเดิม
- **EP 4:** **Text Blocks:** เขียน SQL/JSON ใน Java ให้สวยเหมือนใช้ Notepad++
- **EP 5:** **Records:** บอกลา Getter/Setter และ Lombok ไปตลอดกาล
- **EP 6:** **Sealed Classes:** ล็อกชั้นความลับ! คุมการสืบทอด Class แบบที่ Java 8 ทำไม่ได้
- **EP 7:** เจาะลึก Memory ยุคใหม่: ทำความรู้จัก ZGC ที่แรงจนแทบไม่ต้องจูน
- **EP 8-10:** Access Control 2026: ปกป้องโค้ดให้แน่นหนาฉบับ Senior Dev

### 🧠 Module 2: Logic เทพๆ แบบไม่ต้องพึ่ง if-else (Flow & Patterns)

*เน้นความฉลาดของภาษาที่ช่วยลด Bug*

- **EP 21:** **Modern Switch:** เปลี่ยน Switch แบบเดิมที่น่าเบื่อ ให้กลายเป็นเครื่องจักรคืนค่า
- **EP 22:** บอกลา `break;` รัวๆ ด้วย Arrow Syntax `>` (สะอาดตาขึ้น 10 เท่า)
- **EP 23:** **Pattern Matching:** เช็ค Type พร้อมดึงค่าในบรรทัดเดียว (Smart Casting)
- **EP 24:** การแกะข้อมูลจาก Record: เทคนิค Deconstruction ที่โปรแกรมเมอร์ต้องรู้
- **EP 25:** **Guarded Patterns:** ใส่เงื่อนไข `when` ใน Switch ให้ Logic จบในที่เดียว
- **EP 26-30:** ทบทวน Operators: `++i` vs `i++` และ Logic Gates ที่หลายคนชอบตกม้าตายตอนสัมภาษณ์

### 📝 Module 3: จัดการ Data และความผิดพลาดแบบมือโปร (Strings & Exceptions)

*เน้นงานสาย Backend และ Enterprise*

- **EP 41:** **String Templates:** วิธีฝังตัวแปรลงในข้อความแบบ "ปลอดภัยและเท่"
- **EP 42:** จัดการ SQL Query ยาวๆ ใน Oracle/Postgres ให้แก้รักษาง่าย
- **EP 43:** IO vs NIO.2: เลือกใช้ยังไงให้ระบบไม่อืดเมื่อเจอไฟล์ใหญ่
- **EP 44:** **Modern Exception:** เลือก Exception ให้ตรงงาน (อย่าแค่ `try-catch Exception e`)
- **EP 45:** **Scoped Values:** วิธีส่งข้อมูลข้าม Thread แบบใหม่ที่เบากว่า `ThreadLocal`

### 📚 Module 4: Collections & Generics ระดับ Architect

*เนื้อหาเข้มข้นสำหรับการออกแบบระบบสเกลใหญ่*

- **EP 61:** **Sequenced Collections:** มาตรฐานใหม่ในการจัดการ "ตัวแรก-ตัวสุดท้าย" ของ List/Set/Map
- **EP 62:** ArrayList vs LinkedList: ปี 2026 เรายังควรใช้ตัวไหน? (เจาะลึก Performance)
- **EP 63:** HashMap ทำงานยังไง? ทำไมถึงเป็นคำถามยอดฮิตตอนสัมภาษณ์งาน
- **EP 64:** Generics ขั้นสูง: เข้าใจ Wildcards (`? extends`, `? super`) แบบไม่งงอีกต่อไป
- **EP 65:** เลือก Collection ให้ถูกงาน: ใครเด่นเรื่อง Sorting ใครเด่นเรื่อง Search?

### ⚡ Module 5: Thread ล้านตัว & Spring Boot 4 (The Ultimate Power)

*จุดพีคของซีรีส์: ระบบ High-Scale และการทำงานจริง*

- **EP 81:** **Virtual Threads:** สร้าง Thread เป็นล้านใน RAM แค่นิดเดียว (Project Loom)
- **EP 82:** ทำไมปี 2026 เราต้องเลิกใช้ "Thread Pool" (ExecutorService)?
- **EP 83:** วิธีแก้ปัญหา **Deadlock** และ Race Condition ในยุค Virtual Threads
- **EP 84:** **Structured Concurrency:** จัดการงานขนานแบบ "ไปพร้อมกัน พังพร้อมกัน"
- **EP 85:** **Custom Annotations:** สร้าง "ป้ายกำกับ" ของตัวเองเพื่อทำระบบให้ฉลาดขึ้น
- **EP 90:** Spring Boot 4 + Java 25: สูตรสำเร็จการทำ Microservices ยุคปัจจุบัน
- **EP 100:** **Final Project:** สร้าง API จัดการแคลอรี่ (Next.js + Spring Boot 4 + Postgres)
